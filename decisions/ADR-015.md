The 8-week GTM plan is executed one week at a time, each as an independent /ce:work scope. This keeps each session focused, avoids burning credits planning work that's weeks away, and allows the plan to adapt based on results from prior weeks. Giles completes his manual tasks (Stripe, Pixel IDs, etc.) first, then Claude runs through all code tasks in a single pass with real values — no placeholders.
