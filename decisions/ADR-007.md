A zero-dependency custom useSEO hook for managing page meta tags instead of react-helmet-async. The pre-renderer captures the DOM after React renders, so meta tags set via direct DOM manipulation are picked up in the static HTML output. Avoids adding a runtime dependency for something the build process handles. The hook sets title, description, canonical URL, OG tags, and JSON-LD structured data.
