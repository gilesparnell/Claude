title: Stripe Payment Links over embedded checkout

Pricing page CTAs link to Stripe Payment Links rather than embedding Stripe Checkout or building a custom payment flow. Simpler to set up, no backend required, Stripe handles the full checkout UX including tax collection. Each tier (Starter, Business, Professional) and billing interval (monthly, annual) gets its own Payment Link URL. Trade-off: less control over the checkout experience, but appropriate for current scale and avoids premature backend complexity.
