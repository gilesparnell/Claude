Marketing site and future customer dashboards built as a Vite + React SPA with React Router, not Next.js. GHL is the backend — we're building a frontend that talks to GHL's API, not a full-stack app. Pre-rendering covers SEO for the ~15-30 public pages. Customer dashboards are authenticated SPAs that don't need SSR.

Migration path: if content marketing scales beyond pre-rendering limits (50+ dynamic SEO pages), migrate to Next.js. All React components transfer directly. A headless CMS on a blog subdomain is also viable without migrating.
